Semantic CSS

InstantCSS

easy & rapid & intuitive
In the old way, whenever you needed to style the web, you had to name a class and write CSS.
<head>
    <style>
        .content-container {
            display: flex;
            height: 10em;
            align-items: center;
            justify-content: center;
        }
        .content-box {
            display: flex;
            width: fit-content;
            background: white;
            color: black;
            padding: 0.5em;
            border-radius: 0.5em;
            box-shadow: 0em 1em 1em gray;
         }
        .content-icon {
            width: 4em;
            height: 4em;
            transform: rotate(-15deg);
        }
        .content-subject {
            font-weight: bold;
            font-size: 2em;
            font-variant: small-caps;
        }
        .content-description {
            position: relative;
            margin-top: -0.5em;
            width: fit-content;
        }
        .content-description span {
            font-family: 'Times New Roman', Times, serif
        }
    </style>
</head>
<body>
    <div class="content-container">
        <div class="content-box">
            <img class="content-icon" src="/instant-css/favicon.png" alt="favicon"></img>
            <div>
                <h3 class="content-subject">InstantCSS</h3>
                <div class="content-description">
                    <span>easy & rapid & intuitive</span>
                </div>
            </div>
        </div>
    </div>
</body>
Utillity-First CSS

InstantCSS

easy & rapid & intuitive
With InstantCSS, you style elements by applying pre-existing classes directly in your HTML.
<head>
    <script src="/instant-css.js"></script>
</head>
<body>
    <div class="flex h=10 ai=center jc=center">
        <div class="flex w=fit-content bg=white c=black p=0.5 br=0.5 bs=0em_1em_1em_gray">
            <img class="w=4 h=4 tf=rotate(-15deg)" src="/instant-css/favicon.png" alt="favicon"></img>
            <div>
                <h3 class="bold fs=2 fv=small-caps">InstantCSS</h3>
                <div class="relative mt=-0.5 w=fit-content _span/ff='Times_New_Roman',Times,serif">
                    <span>easy & rapid & intuitive</span>
                </div>
            </div>
        </div>
    </div>
</body>
