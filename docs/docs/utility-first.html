Semantic CSS

			
InstantCSS

				
Near-Atomic CSS Library
In the old way, whenever you needed to style the web, you had to name a class and write CSS.
<head>
	<style>
		.content-container {
			display: flex;
			height: 10em;
			align-items: center;
			justify-content: center;
		}
		.content-box {
			display: fit-content;
			width: 10em;
			background: #fff;
			color: #000;
			padding: .5em;
			border-radius: .5em;
			box-shadow: 0 1em 1em gray;
		}
		.content-icon {
			width: 4em;
			height: 4em;
			transform: rotate(-15deg);
		}
		.content-subject {
			font-weight: bold;
			font-size: 2em;
			font-variant: small-caps;
		}
		.content-description {
			position: relative;
			margin-top: -1.5em;
			width: fit-content;
		}
		.content-description {
			font-family: 'Times New Roman', Times, serif;
		}
	</style>
</head>
<body>
	<div class="content-container">
		<div class="content-box">
			<img class="content-icon" src="/favicon.png" alt="favicon" />
			<div>
				<h3 class="content-subject">InstantCSS</h3>
				<div class="content-description">
					<span>easy & rapid & intuitive</span>
				</div>
			</div>
		</div>
	</div>
</body>
Utillity-First CSS

			
InstantCSS

				
Near-Atomic CSS Library
With InstantCSS, you style elements by applying pre-existing classes directly in your HTML.
<head>
	<script src="/instant-css.js"></script>
</head>
<body>
	<div class="flex h=10 ai=center jc=center">
		<div class="flex w=fit-content bg=#fff c=#000 p=.5 br=.5 bs=0_1_1_gray">
			<img class="w=4 h=4 tf=rotate(-15deg)" src="/instant-css/favicon.png" alt="favicon" />
			<div>
				<h3 class="bold fs=2 fv=small-caps">InstantCSS</h3>
				<div class="relative mt=-1.5 w=fit-content _span/ff='Times_New_Roman',Times,serif">
					<span>easy & rapid & intuitive</span>
				</div>
			</div>
		</div>
	</div>
</body>
