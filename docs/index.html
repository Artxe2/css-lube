How to use

사용하고 있는 JS 라이브러리와 상관없이 HTML 파일에 instant-css를 스크립트 태그로 추가하세요.
instant-css가 자동으로 적용됩니다.

<body>
    ...
    <script src="./instant-css.js"></script>
</body>

Basic

instant-css를 사용하면 클래스명 그대로 스타일에 적용해줍니다.
클래스명에는 공백을 사용할 수 없으므로 공백을 입력해야할 경우 "_"로 대신합니다.
클래스명이 그대로 스타일에 적용되기 때문에 ";"를 사용하면 클래스명 하나에 여러 스타일을 적용할 수도 있습니다.
기본 문법을 활용하면 기능의 제약 없이 그 어떤 스타일이라도 클래스명에 적용이 가능합니다.

Abbreviation

instant-css는 개발 속도 향상을 위해 여러가지 방식의 약어를 지원합니다.
자세한 명세서는 document 페이지에서 확인이 가능합니다.

":" 대신 "=" 사용 가능
대문자 사용시 "-" + 소문자로 치환
자주 쓰는 스타일 이름
이름이 필요없는 스타일 값
일부 스타일 사용시 "px" 생략
calc 함수 사용시 공백 생략
"--" 사용시 var 함수 생략

Selector & Pseudo Class

"/"를 이용하면 선택자로 하위 객체에 스타일을 적용할 수 있습니다.
가상 클래스 역시 같은 방식으로 스타일을 적용할 수 있습니다.

Priority

기본 문법으로 직접 스타일에 important를 선언해줄 수도 있지만
instant-css에서는 클래스명 뒤에 !를 붙이는 것으로 우선순위를 높일 수 있습니다.

<style>
    ...
    [class][class][class].c\=yellow\!\!\!{color:yellow}
    [class][class].c\=orange\!\!{color:orange}
    [class].c\=red\!{color:red}
</style>


Media

미디어 쿼리를 적용하기 위해서는 "@" 두 개 사이에 조건문을 작성하면 됩니다.
not 또는 and 조건을 추가하고 싶다면 not 대신 "!", and 대신 "&"를 사용해야하며 ","는 지원하지 않습니다.
범위조건을 사용할 경우에는 "~"를 이용한 약어를 지원합니다.
미디어 쿼리는 스타일시트에서 나중에 추가되기 때문에 "!" 를 붙이지 않아도 우선순위가 더 높습니다.

<style>
    ...
    .bg\=red{background-color:red}
    @media (max-width: 800px) {
        .\@maxWidth\=800\@bg\=blue{background-color:blue}
        .\@maxWidth\=800\@c\=white{color:white}
    }
</style>

Dark mode

기본적으로 다크모드는 미디어쿼리의 "prefers-color-scheme" 옵션을 사용해서 구현합니다.
!!window.localStorage.getItem("THEME")가 true이고 그 값이 "DARK"가 아니라면 기기 설정과 상관없이 다크모드가 적용되지 않습니다.
!!window.localStorage.getItem("THEME")가 true이고 그 값이 "DARK"라면 기기 설정과 상관없이 다크모드가 적용됩니다.
<script>
    ...
    localStorage.setItem("THEME", "LIGHT") // LIGHT MODE
    localStorage.setItem("THEME", "DARK") // DARK MODE
    localStorage.removeItem("THEME") // SYSTEM MODE
</script>
