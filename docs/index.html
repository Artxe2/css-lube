What is CSS Lube

CSS Lube is Highly-optimized JIT CSS Generator.
This makes improved your developer experience by implement any designs directly in markup and immediately reflect feedback.
In addition, CSS Lube parses HTML documents at runtime and render styles, so it can completely replace style files that become bloated whenever updated with a 6,611 byte js file.
