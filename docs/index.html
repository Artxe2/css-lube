What is CSS Lubricant

CSS Lubricant is highly-optimized On-Demand Utility-First Near-Atomic CSS Library.
This makes improved your developer experience by implement any designs directly in markup and immediately reflect feedback.
In addition, CSS Lubricant parses HTML documents at runtime and render styles, so it can completely replace style files that become bloated whenever updated with a 2823 byte js file(based on gzip).
